3. Elegimos la herramienta para el producto final

Texto

Objetivo del uso de la herramienta:


Fomentar la escritura colaborativa en tiempo real, la organización de ideas y la revisión entre iguales, permitiendo que el producto final sea accesible para el docente, las familias y el resto de la clase desde cualquier dispositivo.

 
Justificación según los criterios exigidos
 

docs.google.com/


Google Docs permite: escritura simultánea de varios alumnos, chat integrado para discutir cambios, historial de versiones para recuperar trabajo perdido, comentarios para retroalimentación del docente, inserción de imágenes y tablas, y exportación a PDF o impresión. Todo sin instalar nada.


Contexto y disponibilidad en el centro


El centro educativo dispone de tablets y ordenadores con conexión a internet. Las cuentas de Google Workspace for Education están activadas para el alumnado de 3º a 6º. No requiere descargar software ni pedir permisos especiales a familias.


Edad y características (6º Primaria, 11-12 años)


El alumnado ya ha usado Google Docs en cursos anteriores para trabajos sencillos (escribir un cuento, hacer una lista). Conocen lo básico: escribir, cambiar fuente, insertar imagen. Para este proyecto solo necesitan repasar los comentarios y la opción de "Sugerencias" en lugar de edición directa.
Competencia digital del alumnado
Nivel competencial A2-B1 (inicial-intermedio). Con Google Docs trabajan:
• Creación de contenido digital (redactar, formatear, insertar imágenes)
• Colaboración digital (trabajo en grupo sincrónico y asincrónico)
• Gestión de archivos en la nube (organizar en carpetas de Drive)
• Ciberseguridad básica (no compartir con personas externas, usar la cuenta del centro)


Opciones de accesibilidad


Google Docs incluye herramientas de accesibilidad muy útiles para 6º de Primaria:
• Compatible con lectores de pantalla.
• Escritura por voz (dictado) para alumnado con dificultades motoras o de escritura
• Ampliación de zoom y alto contraste
• Teclas de acceso rápido (atajos)
• Posibilidad de añadir texto alternativo a las imágenes
Uso selectivo y adaptado al contexto
No se usa Docs de forma genérica. Se adapta así:
• El docente crea un documento base por grupo y lo comparte como "Puede editar"
• Se activa "Mostrar sugerencias" para que las correcciones sean visibles
• Se limita el número de alumnos por documento (máximo 4) para evitar caos
• Se entrega una guía visual con los 5 pasos básicos (imprimible y en PDF)
